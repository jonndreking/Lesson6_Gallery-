2.  List the six main objects that make up the browser object model. 

window.navigator	Information about the browser itself.
window.screen	The user's screen.
window.history	URLs visited by the user.
window.location	The current URL.
window.framename	A frame that occupies the browser window. 
window.document	The document appearing in the main browser window.
